# QuickFarm

## How to Run ?
	- install mongodb dependencies :  
    `npm install --save mongoose mongodb`
	- to test whether the mongodb and mongoose are working properly  
    run 'node mongodb-test.js` 
  - to run the app 
    `npm start`
